# spotify-tableau-dashboard

## 📌 Overview
This project analyzes Spotify dataset using Tableau to explore music trends, including artist performance, album insights, and song popularity.

The goal is to build interactive dashboards that help understand how different factors influence music popularity.

---

## 📊 Dashboards Created

### 🎤 Dashboard 1: Overview Analysis
- Total songs, artists, and albums
- Album performance based on popularity
- General dataset insights
- Song-level popularity analysis

### 🎧 Dashboard 2: Artist & Album Analysis
- Top artists by number of songs
-  Loudness By Artist


---

## 🛠 Tools Used
- Tableau
- Excel / CSV Dataset

---

## 📈 Key Insights
- A small number of artists contribute a large portion of songs
- Song popularity varies significantly across the dataset
- Album performance is influenced by individual track popularity
